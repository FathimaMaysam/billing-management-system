# Billing Management System

A Java console application for managing products, customers, and billing.

## Features
- Add products and customers
- Generate bills with automatic total calculation
- View all bills with customer and product details

## Tech Stack
- Java (OOP, JDBC)
- MySQL (DDL, DML, Joins, Foreign Keys)

## Setup
1. Create database: `billing_db`
2. Run SQL schema from `schema.sql`
3. Update password in `DBConnection.java`
4. Compile: `javac -cp mysql-connector-j-9.7.0.jar *.java`
5. Run: `java -cp .;mysql-connector-j-9.7.0.jar Main`
